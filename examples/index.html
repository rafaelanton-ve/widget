Bienvenido a Social Widget

Instalación

Agrega el siguiente script a tu sitio web:

<script src="https://tu-dominio.com/widget.js" data-tenant="tu-tenant-id"></script>

Características

  • Totalmente aislado con Shadow DOM
  • Estilo visual minimalista estilo Vercel
  • Configuración dinámica desde Supabase
  • Animaciones suaves (250ms ease-out)
  • Iconos SVG embebidos
  • Diseño responsivo

Personalización

El widget se configura desde el panel de administración. Los clientes pueden personalizar:

  • Colores primarios y secundarios
  • Logo de la empresa
  • Descripción
  • Enlaces a redes sociales
  • Enlaces adicionales
  • Posición del widget
Nota: El widget requiere configuración de Supabase. Reemplaza las credenciales de API en el código antes de usar en producción.
